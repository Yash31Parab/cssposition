POSITION

  • relative :- It is the property of the element which allows to move the element wrt. its root position

  • absolute :- Position static is default properties of a element.

  • sticky :- It is the property which combines position relative and sticky.The only thing is when it satisfy a CSS property its property starts working

  • fixed :- It is the property which allows the element to remained fixed in that particular position when the page scrolls

  • static :- Position static is default properties of a element.Here Top,left,right and bottom properties wont work
